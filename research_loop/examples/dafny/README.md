# Dafny examples

Small standalone `.dfy` files used for experiments and manual verification.

| File | Purpose |
|---|---|
| `query1_extern.dfy` | Bounded native newtypes for SSB Q1-style scan; used by `scripts/extern_demo.sh` |
| `test_q7.dfy` | Minimal SSB Q7 `MethodSpec` snippet |
| `test_full_q7.dfy` | Fuller Q7 spec variant for manual Dafny checks |
